# Utility Hub

A simple, modern utility website with 8 handy tools in one place.

## Tools Included

- **Calculator** — Basic math operations
- **Random Generator** — Numbers (min-max), dice rolls (e.g. 2d6), passwords
- **Time Converter** — Convert between timezones
- **Gemini AI** — Chat with Google's Gemini (requires API key from [Google AI Studio](https://aistudio.google.com/apikey))
- **Unit Converter** — Length, weight, temperature, volume
- **Color Picker** — Pick colors, view HEX/RGB values
- **Base64** — Encode and decode text
- **JSON Formatter** — Format and minify JSON

## How to Run

Open `index.html` in your browser — no server required. Or use a simple local server:

```bash
npx serve .
```

Then visit http://localhost:3000

## Gemini API

Get a free API key at [aistudio.google.com/apikey](https://aistudio.google.com/apikey). Your key is only used in your browser and never sent anywhere except Google's API.
